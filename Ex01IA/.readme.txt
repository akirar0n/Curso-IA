# 📊 Análise Estatística e Modelo de Previsão
📌 Sobre o Projeto

Este repositório contém a solução do Exercício 01 – Análise Estatística e Modelo de Previsão, desenvolvido para a disciplina de Inteligência Artificial do curso de Sistemas de Informação.
O projeto simula a atuação de um analista de dados, utilizando conceitos básicos de estatística e lógica de previsão para avaliar o desempenho de alunos a partir de fatores de estudo e frequência.

🎯 Objetivo

    Gerar uma base de dados simulada com 10 alunos

    Calcular estatísticas descritivas (média e desvio padrão)

    Criar um índice de aproveitamento (IAp)

    Classificar os alunos quanto ao risco de reprovação

    Realizar uma análise crítica sobre o modelo proposto
